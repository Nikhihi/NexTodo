合并说明
这个文件把桌面、平板、手机原型合并为一个入口。切换设备时不会丢失任务状态，适合你后面照着拆 QML 组件和响应式布局。
布局映射
桌面：顶部工具栏 + 筛选排序 + 列表详情双栏 + 底部统计。
平板：横屏双栏，保留列表与详情同时可见，减少弹跳导航。
手机：单列列表，点击任务进入详情页，返回键回列表。
新增/编辑：桌面和平板居中弹窗，手机底部表单。
日期分组：已过期、今天、明天、以后、无日期、已完成。
